# Frontal dynamics and phytoplankton decoupling in the Alboran Sea from high-resolution satellite data

**Status:** Published
**Start date:** 2020-10-01
**Accepted date:** 2026-03-10
**Authors:** Iria Sala, Isabel Caballero, Marina Bolado-Penagos, Gabriel Navarro

## Summary
This study analyses the seasonal and interannual variability of sea surface temperature (SST) and chlorophyll-a (chl-a) fronts in the Alboran Sea using 1-km resolution satellite data.
Fronts are detected using the Canny Edge Detection (CED) algorithm, from which two complementary metrics are derived: frontal gradient (FG) and frontal probability (FP). 

The analysis reveals a strengthening of SST-derived fronts over time, while chlorophyll fronts exhibit higher variability and long-term decline, indicating a progressive decoupling between physical frontal structures and phytoplankton response.
